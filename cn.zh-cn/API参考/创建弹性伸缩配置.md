# 创建弹性伸缩配置<a name="as_06_0201"></a>

## 功能介绍<a name="section66578044"></a>

创建弹性伸缩配置。

-   伸缩配置是伸缩组内实例（弹性云服务器）的模板，定义了伸缩组内待添加的实例的规格数据。
-   伸缩配置与伸缩组是解耦的，同一伸缩配置可以被多个伸缩组使用。
-   默认最多可以创建100个伸缩配置。

## 调试<a name="section948154693415"></a>
